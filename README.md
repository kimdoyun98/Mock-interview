# Mock-interview
모의 면접

<br>

# 프로젝트 소개
취업 준비 중 면접 스크립트를 만들고 질문을 생각하며 허공에 대고 읊조리고 있는 나의 모습을 보고 생각나 만들게 된 모의 면접 애플리케이션입니다.

면접 질의에 대해 스스로를 촬영 후 모니터링하여 단점을 보완하고 개선하여 좋은 결과를 도출할 수 있도록 만든 앱입니다.

<br>


# 개발 기간
 * 24년 02월

<br>


# 핵심 기능
* 앱 내에서 동영상 촬영 후 로컬에 저장
* 면접 질의 별로 구분하여 동영상 목록 구현
* Exoplayer2를 사용하여 동영상 재생
 <br>
 

# Tech Stack
`Kotlin` `MVC` `YoutubePlayer` `Content Provider` `Exoplayer2` `CameraX`

<br>


# Architecture
* `MVC`

<br>


# Learn
* Exoplayer2를 이용한 동영상 재생 기능에 대해 알게됨
* CameraX를 사용하여 앱 내에서 동영상 촬영 방법에 대해 알게됨
